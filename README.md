# DriveEasy – Car Showcase & Test Drive Booking Website

## 📌 Project Overview
DriveEasy is a responsive multi-page car showcase website developed to display newly launched vehicles and allow users to schedule test drives online.

This project was created as a self-learning frontend development project while learning Java Full Stack Development.

The website includes multiple pages such as Home, About Us, Vehicles, Services, and Contact Us with a clean and user-friendly interface.

---

## 🚀 Features
- Responsive website design
- Multi-page navigation
- Vehicle showcase section
- Test drive booking form
- About Us page
- Services section
- Contact Us page
- Interactive UI components
- Mobile-friendly layout
